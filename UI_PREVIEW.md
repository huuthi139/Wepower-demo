## 🧩 Components Detail

### Button Variants
```
Primary:    [🔴 Bắt đầu ngay]         Red gradient + glow
Secondary:  [⚪ Xem demo]              Border transparent
Ghost:      [Đăng nhập]                Transparent hover
Accent:     [Premium]                  Gold gradient
Outline:    [Tìm hiểu thêm]           Red border
```

### CourseCard States
```
Default:    Gray-900 background, border gray-800
Hover:      Border → primary-600, shadow-card-hover, scale image
Active:     (same as hover)

Badge Types:
- BESTSELLER → Gold (accent-400)
- NEW → Red (primary-600)
- PREMIUM → Gradient (accent → primary)
- MIỄN PHÍ → Green
```

### Header States
```
Desktop:  Logo | Links | Buttons
Mobile:   Logo | [☰] → Dropdown menu

Sticky:   bg-black/80 backdrop-blur
Scrolled: border-b border-gray-800
```

## 🔍 Interactive Features

### Search
- Type: Client-side filtering
- Fields: title, instructor
- Case-insensitive

### Category Filter
- Click category button → filter courses
- Active state: Red background + glow
- "Tất cả" → show all

### Sort
- Dropdown with 5 options
- Frontend only (not functional in demo)

### Progress Bar
- Width based on percentage (0-100%)
- Gradient: primary-600 → accent-400
- Animated transition
